# <center>Finance Tracker</center>
#### <center>Простое риложение для учёта доходов и расходов на Python</center>
***
### :gem: Использованные технологии
- PyQt5
- Matplotlib
- SQLite3
- PyInstaller
- Очистка базы данных
***
### :open_file_folder: Архитектура проекта
- Модульная структура
- Классическая MVC-логика
***
### :gear: Функции приложения
- Добавление и редактирование доходов и трат
- Управление категориями с иконками
- Диаграмма расходов
- Импорт и экспорт CSV
- Очистка базы данных

## :package: Установка и запуск
```bash
git clone https://github.com/USERNAME/FinanceTracker.git
cd FinanceTracker
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m app.main
```
